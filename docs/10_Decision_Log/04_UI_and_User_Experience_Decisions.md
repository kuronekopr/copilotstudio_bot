# 04_UI_and_User_Experience_Decisions

## UD-01: マスク削除不可設計

**決定**：自動検出された PII マスクはユーザーが削除できない

**理由**：
- PII 再露出防止（ユーザーミスの場合も）
- システムが最善を尽くした証拠
- 法的責任の明確化

**UI 実装**：自動検出 checkbox は disabled

---

## UD-02: PII マスク確認の強制

**決定**：PII 検出時、ユーザーによるマスク確認画面を必須化

**理由**：
- ユーザーが意図せず PII を送信することを防止
- UI で確認 = 同意の記録
- 法的対応（個人情報保護法）

**実装**：マスク確認なしに「送信」ボタンをクリックしても、確認画面に遷移

---

## UD-03: 3 レベルの判定表示

**決定**：スコアリング結果を 3 段階（AUTO/ASK/ESCALATE）に分類して表示

**理由**：
- シンプルなユーザーメンタルモデル
- スコア数値は隠す（ユーザーは判定のみ）
- 説明責任が容易
