## 14. トーン & マナー

### 基本原則
- **丁寧**：常に敬語を使用
- **簡潔**：不要な表現を避ける
- **明確**：ユーザーの次の操作を示す
- **親切**：エラー原因と解決策を記載

### 例
```
❌ 悪い例：
「エラーです」
「失敗しました」

✅ 良い例：
「申し訳ございません。画像サイズが大きすぎます（最大 5MB）」
「ネットワーク接続を確認してください」
```

### 親しみやすい表現
```
「もう一度お試しください」
「お手数ですが」
「申し訳ございません」
「ご協力ありがとうございます」
「こちらをご参照ください」
```
